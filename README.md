# Projet Gestion d'Hôpital

Ce projet Spring Boot permet de gérer les patients, médecins et rendez-vous d’un hôpital.

## Installation
1. Cloner le dépôt : `git clone https://github.com/ton-nom/ton-projet.git`
2. Ouvrir le projet dans VS Code ou IntelliJ.
3. Lancer avec `mvn spring-boot:run`
